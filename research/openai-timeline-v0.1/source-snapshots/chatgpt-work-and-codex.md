# ChatGPT Work and Codex

Choose Work for research and deliverables, or Codex for software development.

Updated: 2 days ago

ChatGPT Work is gradually rolling out to eligible accounts over the coming days. If you don't see ChatGPT Work in your account yet, your account may not have access yet. Availability will continue to expand as the rollout progresses.

ChatGPT includes Chat, ChatGPT Work, and Codex. Chat handles questions and conversation. Work takes on longer research and creates finished materials. Codex remains the dedicated agent for software development and technical work.

Work follows the same usage structure as Codex. The [Codex pricing page](https://developers.openai.com/codex/pricing) explains included usage and additional credits; its examples are based on coding tasks, so Work usage will vary by task.

# Choose an experience

*   **Chat:** Ask questions, search, brainstorm, or get quick conversational help.

*   **Work:** Research a topic, analyze information, or create a document, spreadsheet, presentation, report, or Site.

*   **Codex:** Write or debug code, run tests and commands, review changes, or work with a repository.

# Where Work and Codex are available

Work is available on ChatGPT web and mobile for eligible paid plans. Work is also available in the ChatGPT desktop app when included for your plan and workspace.

Work on web and mobile runs in the cloud. Work in the desktop app can also use local files and desktop apps with your permission. At launch, cloud Work conversations do not appear in desktop Work; desktop Work threads and local files remain on that computer.

Codex is available as a mode in the ChatGPT desktop app. It can work with local folders, repositories, terminals, and developer tools. Codex is not a selectable mode on web or mobile. You can access supported desktop Codex tasks from the Remote tab in the ChatGPT mobile app, but those tasks do not become web or mobile chat history.

# Start on desktop

1.   Open the ChatGPT desktop app and sign in.

2.   Start a standalone chat, or open a project or local folder for ongoing work.

3.   Select Work or Codex from the mode switcher.

4.   Describe the outcome you want. Add any files, constraints, and review criteria ChatGPT should use.

5.   Review the result, then ask for changes or follow-up work in the same conversation.

# Start Work on web or mobile

1.   Open ChatGPT on the web or in the mobile app and select Work.

2.   Start a new conversation or open an existing project.

3.   Add any files and context ChatGPT should use, then describe the task or deliverable you want completed

# Keep work moving across devices

A project keeps related conversations, files, and instructions together. Work can also run once, repeat on a schedule or trigger, or monitor for changes through Scheduled Tasks. You can review progress, answer questions, change direction, and approve important actions.

Chat conversations sync across ChatGPT web and the desktop app. Work conversations created on web or mobile stay in the cloud and can continue on supported cloud surfaces. At launch, they do not appear in desktop Work. Desktop Work threads and local files remain on that computer. Codex desktop tasks do not appear on web; use the Remote tab in the ChatGPT mobile app to access supported remote tasks.

# FAQ

## Can Work use files on my computer?

Yes, in the desktop app. Open a local folder or project and grant access only to the files the task needs. Local files and outputs remain on that computer unless you explicitly move or share them. Work on web and mobile cannot directly access files on your computer.

## Is Codex available on web or mobile?

Codex is not a selectable mode on web or mobile. You can access supported desktop Codex tasks from the Remote tab in the ChatGPT mobile app. Remote access does not add those tasks to your web or mobile chat history.

## Where can I learn more?

See Using Codex with your ChatGPT plan for Codex access and usage. See Projects in ChatGPT for help organizing ongoing work, and Scheduled Tasks in ChatGPT for recurring or trigger-based work.

## How do you switch between Chat, Work, and Codex?

**On desktop:**

You can select between Work and Codex by clicking “ChatGPT Work” or “ChatGPT Codex” in the top left corner of the ChatGPT Desktop app.

To start a new Chat, click the new “Quick chat” button in the left-hand navigation bar.

**On mobile:**

You can select between Work and Chat by clicking the dropdown (showing “Chat” or “Work”) at the top of the screen.

You can’t select Codex on mobile, but you can access supported desktop Codex tasks from the Remote tab in the ChatGPT mobile app.

## What’s happening to regular ChatGPT?

The fast, familiar experience you already know is still available.

On desktop, to start a new Chat, click the new “Quick chat” button in the left-hand navigation bar.

On mobile, just select "Chat" from the dropdown (showing “Chat” or “Work”) at the top of the screen.
